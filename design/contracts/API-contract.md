## 2. 端点定义

#### 请求

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| （参数1） | （string/int/bool） | 是 | （描述） |

#### 响应 - 成功 (200)

```json
{
  "status": "success",
  "data": {
    "（字段1）": "（类型）"
  }
}
```

#### 响应 - 错误

| 状态码 | 说明 |
|--------|------|
| 400 | 请求参数错误 |
| 401 | 未认证 |
| 403 | 无权限 |
| 500 | 服务器错误 |
